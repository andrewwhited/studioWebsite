# Spacing System

## Base Unit

The spacing system derives from the **4px base unit** used across the
design system.

All spacing values must be multiples of 4px.

## Spacing Scale

-   xs: 4px
-   sm: 8px
-   md: 12px
-   lg: 16px
-   xl: 24px
-   xxl: 32px
-   xxxl: 40px
-   section: 48px
-   large-section: 64px

## Usage

Spacing tokens should be used instead of arbitrary spacing values.

Spacing tokens apply to:

-   margin
-   padding
-   gap
-   grid gutters
-   component spacing

## Principles

-   Spacing reinforces grouping and hierarchy.
-   Smaller spacing is used inside components.
-   Larger spacing separates layout sections.
-   All spacing aligns to the 4px base unit.
