# Typeface

Active Grotesk, served via Adobe Fonts.

## Weights

Light, Regular, Medium.

## Scale and Grid

The type scale is the minor third (1.2), and the baseline grid is 4
pixels.

## Sizes

Base size is 14 (1 rem).

-   Small: 0.833rem, line-height 16px
-   Paragraph: 1rem, line-height 20px
-   H6: 1.2rem, line-height 24px
-   H5: 1.44rem, line-height 28px
-   H4: 1.728rem, line-height 32px
-   H3: 2.074rem, line-height 36px
-   H2: 2.488rem, line-height 40px
-   H1: 2.985rem, line-height 48px
