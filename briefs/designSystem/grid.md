# Layout Grid

## Base Unit

The layout system is based on a **4px base unit**.

All spatial measurements in the interface should derive from multiples
of this unit.

## Baseline Grid

Typography aligns to a **4px baseline grid**.

Line heights and vertical spacing should resolve to multiples of the
base unit.

## Columns

The layout uses a **12-column grid**.

This allows the following common layout divisions:

-   Full width: 12 columns
-   Half width: 6 columns
-   Third width: 4 columns
-   Two thirds: 8 columns
-   Quarter width: 3 columns

The layout favors **halves and thirds** as the primary compositional
structure.

## Gutters

Gutters should use multiples of the base unit.

Recommended default: - 24px (6× base unit)

## Responsive Behavior

Suggested column behavior:

Desktop: 12 columns\
Tablet: 6 columns\
Mobile: 4 columns
