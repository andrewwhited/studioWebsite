# Design Reference Pool

A shared pool of market and inspiration references for both the main site and UX site critiques. Each entry is tagged by what it exemplifies. When conducting a critique, select 4ŌĆō5 references most relevant to the lowest-scoring or most active categories that session.

## How to Select References

Match references to the session's focus areas:

| If weakest in...          | Pull references tagged with...     |
|---------------------------|------------------------------------|
| Voice / Content           | `voice` `content` `editorial`     |
| IA / Navigation           | `ia` `navigation` `structure`     |
| Typography                | `typography` `type`               |
| Grid / Layout             | `layout` `grid`                   |
| Market Positioning        | `positioning` `seniority`         |
| Design Leadership         | `leadership` `seniority`          |
| Restraint / Authored Feel | `restraint` `authored`            |
