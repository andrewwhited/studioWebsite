# Studio Website

A multi-site workspace for Andrew Whited’s web presence.

## Overview

This repository contains the source materials and site code for two related but distinct web properties:

- a main studio site
- a dedicated UX portfolio site

## Repository Structure

- `briefs/` — source documents, content briefs, IA notes, and project guidance
- `main/` — main studio/personal website
- `ux/` — UX portfolio site

## Notes

The `briefs` directory is the source of truth for strategy, structure, voice, and content planning across both sites.
